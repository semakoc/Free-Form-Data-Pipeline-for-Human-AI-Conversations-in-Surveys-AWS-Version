# Free-Form Chatbot Pipeline for Surveys (AWS Version)

A production-ready web chatbot pipeline designed for embedding conversational AI into surveys and securely logging interaction data to AWS.
This version is built for research and institutional deployment using AWS infrastructure. It supports embedding in survey platforms such as Qualtrics and logs conversations to AWS S3 for structured storage and later analysis.

## Overview

This project provides:

- A lightweight web chatbot interface  
- A Flask backend connected to OpenAI  
- Secure logging of chat data to AWS S3  
- Public access via Cloudflare Tunnel  
- Survey embedding support through iframe integration  

This version is intended for secure, scalable deployment.
